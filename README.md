# 前端client

vue3 + vite 

```中文
需安装
* axios
* pinia  状态管理模块
* sass  简化css
* vue-router  路由
* naive-ui  ui
* wangEditor 富文本编辑
```

启动，在client目录下执行 npm run dev

# 后端server

数据库sqlite3 + node.js

```
* multer  处理上传功能的中间件
* sqlite3 数据库
* uuid  不重复的唯一标识
```

启动，在server目录下执行 node app.js
